# Niyantran Dashboards

## Overview
This project contains three separate dashboard interfaces:

- BESCOM Dashboard  
- EV User Dashboard  
- Infrastructure Planning Dashboard  

Each dashboard is provided as an HTML file and can be opened directly in a web browser.

---

## Files Included

- bescom_dashboard.html  
- ev_user_dashboard.html  
- infrastructure maker_dashboard.html  

---

## How to Download

### Option 1
Download the entire repository

- Click the green Code button  
- Click Download ZIP  
- Extract the ZIP file  

### Option 2
Download individual files

- Open the file you want  
- Click Download or right click and save  

---

## How to Open the Dashboards

### Method 1
- Locate the downloaded HTML file  
- Double click the file  
- It will open in your default browser  

### Method 2
- Right click the HTML file  
- Select Open with  
- Choose Chrome, Edge, Firefox, or Safari  

### Method 3
- Open a web browser  
- Drag the HTML file into the browser window  

---

## Notes

- No installation required  
- Works on any modern browser  
- Internet connection may be required if external resources are used  

---

## Purpose of Each Dashboard

- BESCOM Dashboard  
  Grid monitoring and load management  

- EV User Dashboard  
  Charging control and user preferences  

- Infrastructure Dashboard  
  Planning and expansion insights
